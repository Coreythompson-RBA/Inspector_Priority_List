MARS Data Converter

Turns your MARS workbook into the data.json file the Inspector Priority List site needs. Everything happens in your browser — your file is never uploaded anywhere.

Drop your workbook here
or click to choose a file (.xlsx or .xlsm)
How to use this
  1. Save/have a copy of your MARS workbook (the one with the "MARS" tab).
  2. Drop it above — you don't need to export or select columns first, this reads the "MARS" sheet directly.
  3. When it finishes, a file called data.json downloads automatically.
  4. Go to your GitHub repo, open data.json, use "Upload files" to replace it with the new one you just downloaded, and commit.
